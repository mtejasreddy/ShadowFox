### 📌 Task 2: Simple Contact Management System

## Problem Statement

Create a command-line contact management system that allows:

- Add new contacts (Name, Phone, Email).
- View all contacts.
- Update existing contacts (by index).
- Delete contacts (by index).

# Key Learnings

- `CRUD operations` in Java
- Using ArrayList to store objects
- User input/output handling with `Scanner`

# How to Run

1. Compile the program:
   - javac ContactManagementSystem.java

2. Run the program:
   - java ContactManagementSystem

3. Use the menu options to manage contacts.
