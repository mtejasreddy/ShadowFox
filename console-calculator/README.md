# ShadowFox Internship - Beginner Tasks

## 📌 Task 1: Enhanced Console-Based Calculator

### Problem Statement

Develop a console-based calculator that supports:

- Basic arithmetic (Addition, Subtraction, Multiplication, Division)
- Scientific calculations (Square Root, Exponentiation)
- Unit conversions (Temperature: Celsius <-> Fahrenheit, Currency: INR <-> USD)

### Key Learnings

- Control flow using `switch` statements
- Exception handling (`try-catch` for invalid input/division by zero)

### How to Run

1. Compile the program(Terminal):
   - javac Calculator.java

2. Run the program:
   - java Calculator

3. Follow the menu prompts to perform operations.
