# Portfolio Website

This is my personal portfolio project showcasing my skills, projects, and resume.  
It is built using **HTML, CSS, and JavaScript** with a clean, responsive design.

---

│── assets/ # Resume, documents, and other static files

## How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/mtejasreddy/ShadowFox.git
   cd ShadowFox/Portfolio

   ```

2. For static HTML/CSS/JS:
   Open the project:

   ```bash
   explorer.exe index.html # Windows
   open index.html # macOS
   ```
