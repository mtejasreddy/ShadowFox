# Mumbai Indians IPL Team Website

A fan based website dedicated to the Mumbai Indians IPL team, built as part of the **ShadowFox Web Development Internship (Intermediate Project)**.  
The site showcases team information, match schedules, player profiles, statistics, news updates, and fan engagement features — all designed with MI’s signature **blue and gold theme**.

---

## Theme & Branding

- **Colors**: Blue (#045DE9), Gold (#FFD700), White (#FFFFFF), Dark Navy (#002147)
- **Tagline**: _“Duniya Hila Denge”_
- **Logo**: Mumbai Indians official logo
- **Style**: Clean, modern, responsive design

---

## Project Structure

- **Home (Hero Banner)**: Team logo, tagline, quick links
- **Players Section**: Grid of player cards (Rohit Sharma, Bumrah, SKY, Hardik Pandya, etc.)
- **Match Schedule**: Table of fixtures with filters (Upcoming / Past)
- **Statistics**: points table, top performers
- **News Section**: Blog-style cards with headlines and “Read More” links
- **Fan Zone**: Polls, comment box, embedded social media feed
- **Footer**: Contact info, social media icons, copyright

---

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Optional Enhancements**:
  - Responsive design with media queries
  - Social media integration (Twitter feed)

---

## Features

- Responsive design for desktop and mobile
- Interactive navigation with smooth scrolling
- Player cards with hover effects
- Match schedule filtering
- Fan engagement (polls + comments)
- Social media integration
